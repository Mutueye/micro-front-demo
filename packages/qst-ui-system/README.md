# qst-ui-system
QST前端UI基础库，主要功能：
- [x] 主题变量配置：颜色/字色/背景色/圆角/间距/组件尺寸 等等，作为UI设计和前端开发之间的“接口”
- [x] 全局初始化样式：normalize，全局文字配置(字体/抗锯齿等)
- 组件库适配主题变量配置(样式覆盖)
  - [x] element-plus
  - [ ] element-ui
  - [ ] ant-design / ant-design-vue
  - [ ] vant
- css框架适配主题变量配置
  - [x] unocss
  - [ ] windicss
  - [ ] tailwind
- [ ] 整合夜间模式切换功能
